# コマンド一覧

- `/warps` - 別のワールドへ移動する。
- `/spawn` - ワールドスポーン地点へ移動する。
- `/sethome [名称]` - ホーム地点を設定する。
- `/delhome [名称]` -ホーム地点を削除する。
- `/homes` - ホーム地点一覧を表示する。
- `/bal` - 現在所持しているゲーム内通貨を表示する。
- `/baltop` - ゲーム内通貨の所持ランキングを表示する。
- `/afk` - AFK状態を切り替える。
- `/ignore [プレイヤー名]` - プレイヤーからの通知を無視する。
- `/msg [メッセージ] [プレイヤー名]` - プレイヤーへプライベートチャットを送信する。
- `/playtime` - プレイ時間を確認する。
- `/playtimetop` - プレイ時間ランキングを表示する。
- `/ping` - PING値を表示する。
- `/sit` - その場に座る。
- `/tpa [プレイヤー名]` - プレイヤーへテレポートリクエストを送信する。
- `/tpaccept [プレイヤー名]` - プレイヤーから送信されたテレポートリクエストを承認する。
- `/tpdeny [プレイヤー名]` - プレイヤーから送信されたテレポートリクエストを拒否する。
