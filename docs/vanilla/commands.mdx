# コマンド一覧

- `/warps` - ワープメニューを開く。
- `/market` - マーケットメニューを開く。
- `/skills` - スキルメニューを開く。
- `/spawn` - スポーンワールドへ移動する。
- `/homes` - ホーム地点一覧を表示する。
- `/home [名称]` - ホーム地点へ移動する。
- `/sethome [名称]` - ホーム地点を設定する。
- `/delhome [名称]` -ホーム地点を削除する。
- `/afk` - 放置状態を切り替える。
- `/msg [メッセージ] [プレイヤー名]` - プレイヤーへプライベートチャットを送信する。
- `/ping` - PING値を表示する。
- `/tpa [プレイヤー名]` - プレイヤーへテレポートリクエストを送信する。
- `/tpaccept [プレイヤー名]` - プレイヤーから送信されたテレポートリクエストを承認する。
- `/tpdeny [プレイヤー名]` - プレイヤーから送信されたテレポートリクエストを拒否する。
