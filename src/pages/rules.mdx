# ルールについて

私達のサーバでは可能な限りルールを最低限に保つように心がけています。
しかしながら、不特定多数のプレイヤーが参加している為に、従う必要のあるルールがいくつか存在しています。

ルールの抜け穴を探すことは全く賢くなく、楽しく遊ぶための場所が無くなるだけです。

もしも、ルールについて疑問があればDiscordからサポートチケットを作成してください。

---

プレイヤー、ランク、その他の要素に関わらず、記載されている全てのルールは全てのプレイヤーに適用されます。

スタッフは常に以下の権利を有します。

1. 事前の通知なしにルールを変更する権利
2. 例外を認める権利
3. 独自の裁量でルールを執行する権利

# 1. 全てのプレイヤーを尊重してください

* 不快感を与えたり、他人を見下したりしないでください。
  * スパム行為、悪口や不適切な言葉の使用、他のプレイヤーへの侮辱や嫌がらせ、荒らし等はミュートになることがあります。
  * いかなる状況であっても、中傷的な言葉、侮辱的な言葉、嫌がらせ、悪意のある言葉、差別的な言葉、わいせつな言葉、これらに類する言葉を使わないでください。
* 冒涜的、扇動的、嫌がらせ的な言葉を使わないでください。
  * 中立的で誰にも不快感を与えない言葉を選んで使ってください。
* 個人情報を共有しないでください。
  * 個人情報とは、IP、氏名、メールアドレス、電話番号などです。
  * 誰の個人情報であっても共有しないでください。たとえ自分の個人情報であっても共有しないでください。

# 2. 全ての建築を尊重してください

* 建築する場所を意識してください。
  * 他のプレイヤーの建築から離れた場所に建築してください。
  * 故意に既存の建築の近くに建てたり、保護したりしないでください。
* 盗まないでください。
  * 自身の所有物ではない場合、何かしらの許可を得ていないのであれば、何もしないでください。
  * これには自身のものではない建築を保護することができないことも意味しています。
* 常識的な行動をしてください。
  * 不適切なもの、不快なものを作らないでください。
  * 壊したり、勝手に作ったり、景観を損ねたり、他人の建築の邪魔をする行為は全て禁止です。

# 3. 公正なプレイをしてください‌

* 不正行為をしないでください。
  * ハック、チート、マクロ、その他の不正に利益を得るための行為をしないでください。
  * クライアントのパフォーマンスの変更、描画の変更、HUDの変更、ミニマップ（エンティティとプレイヤーなし）は許可されています。
* 不具合を悪用しないでください。
  * 不具合やその可能性がある行為をしないでください。
  * もしも、発見した場合は報告してください。
* 意図的にサーバをクラッシュさせたり、ラグを発生させないでください。
