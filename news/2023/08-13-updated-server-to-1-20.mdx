バックエンドサーバがMinecraft Java Edition 1.20.1へアップデートされました。

<!-- truncate -->

![Minecraft 1.20](https://i.imgur.com/6Uzen84.jpg)

Minecraft Java Edition 1.20が6月8日にリリースされました。
十分なアップデートの確認やサーバの安定性の確保ができたと判断し、アップデートを実施しました。

1.20へのアップデート、及び、その他の変更に関連する問題が発生する可能性があります。もしも、何らかの問題を発見した場合は、Discordサーバよりご報告ください。
