バックエンドサーバが Minecraft Java Edition 1.21 へアップデートされました。

![Minecraft 1.21](https://i.imgur.com/pV8PGRi.png)

Minecraft Java Edition 1.21 が8月8日にリリースされました。
十分なアップデートの確認やサーバの安定性の確保ができたと判断し、アップデートを実施しました。

Minecraft 1.21 へのアップデート、及び、その他の変更に関連する問題が発生する可能性があります。もしも、何らかの問題を発見した場合は、Discord サーバよりご報告ください。
