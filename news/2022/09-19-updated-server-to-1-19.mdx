---

2022年9月19日のアップデートにより、バックエンドサーバがMinecraft Java Edition 1.19.2へアップデートされました。

![Minecraft 1.19](https://i.imgur.com/jRxbckm.jpg)

Minecraft Java Edition 1.19が6月7日にリリースされていましたが、MCPlayNetworkは安定性やいくつかの問題の観点からバックエンドサーバのアップデートが見送られていました。しかしながら、1.19.2のアップデート以降、安定性が非常に向上し、MCPlayNetwork自体が抱える問題面も一通り解消されましたので、アップデートを実施いたしました。

アップデートに伴い、下記の変更が行われています。

* 資源ワールドの全リセット
* 資源ワールドのリセット周りのシステム変更
* 死亡時にインベントリをロストしないように変更
* 資源ワールドへのテレポート範囲の変更

1.19へのアップデート、及び、その他の変更に関連する問題が発生する可能性があります。もしも、何らかの問題を発見した場合は、Discordサーバよりご報告ください。
